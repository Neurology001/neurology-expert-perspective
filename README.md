## 是什么

一套可在 Claude Code 中激活的神经科临床思维框架，提供以下能力：

- 系统性神经解剖定位推理
- 撒网式鉴别诊断（按病因类别 + 可治性分层）
- 循证挑战既有诊断
- 内置9大临床知识模块（定位速查、CSF 解读、晕厥/癫痫、运动障碍、儿科神经、感染免疫、ALS、28个罕见病影像标志物）

框架以「先定位再定性，先撒网再收网」为核心逻辑，推理过程透明展示。

## 核心心智模型（6个）

| 编号 | 名称 | 一句话 |
|------|------|--------|
| 1 | 定位优先于定性 | 先确定哪里坏了，再想为什么坏 |
| 2 | 系统性鉴别诊断 | 系统排查，按证据和可治性收网 |
| 3 | 循证挑战既有诊断 | 既有诊断是假说，你的工作是检验它；内置锚定/过早收网/可得性偏差识别清单 |
| 4 | 经验模式识别的价值与局限 | 「见过」是提出假说的起点，不是诊断终点 |
| 5 | 概率思维 | 临床规律是概率规律，任何规则都要标注例外概率 |
| 6 | 一元论叙事重建 | 用一个机制解释所有表现，优于多诊断拼凑 |

## 内置知识库

### A. 定位诊断速查（人卫第9版）

- 症状→定位对应表（三偏征、交叉瘫、Wallenberg 综合征等 13 种典型综合征）
- 大脑皮质各叶破坏性/刺激性症状表
- 脊髓节段横向定位表（C1~马尾）
- 共济失调四类型鉴别（小脑性/感觉性/前庭性/额叶性）

### B. CSF 快速解读（人卫第9版）

9 种疾病的脑脊液特征对比（化脓性/结核/病毒/隐球菌/蛛网膜下腔出血/GBS/MS/脑膜癌病/椎管梗阻）+ 关键生物标志物速查（OB、AQP4、MOG、NMDA-R、14-3-3 等）

### C. 问诊要点（人卫第9版 + 临床认知科学）

- 头痛 7 维度
- 呼吸模式定脑干水平
- 瞳孔定位规则
- 疼痛反应（去皮质/去大脑强直）
- **病史可信度主动评估**：5类高漏报领域（药物/毒物、职业暴露、饮酒、性行为史、家族史）各附追问技巧和客观核对方法

### D. 晕厥 vs 癫痫鉴别（Case Files）

- 晕厥与癫痫的临床鉴别特征表
- 晕厥三大亚型（血管迷走性/体位性/心源性）及各自典型表现
- 心源性晕厥高风险因素列表
- 检查路径（ECG / EEG / 心电监测选择逻辑）

### E. PNES（心因性非癫痫性发作）（Case Files）

- 与真性癫痫的临床鉴别（眼睑、运动类型、意识、EEG）
- 诊断（视频EEG为金标准）与处理原则

### F. 运动障碍鉴别（Case Files）

- 特发性震颤 / 帕金森病 / 亨廷顿病 / 迟发性运动障碍四病对比表
- 帕金森综合征"红旗征"（PSP / MSA / CBD / DLB 鉴别）

### G. 儿科神经病学速查（Case Files）

8 个儿科病种：热性惊厥 / BECTS / 儿童偏头痛 / DMD / Tourette / 无脑回畸形 / ASD

### H. 感染性神经病 & 免疫介导神经病（Case Files）

- ADEM / 婴儿肉毒中毒 / 散发性CJD / 脊髓痨 / 脑弓形虫病（五病速查表）
- CIDP vs GBS 鉴别（病程/治疗/预后）
- ALS 诊断框架（El Escorial 标准 + 拟似病鉴别）

### I. 罕见病影像标志物速查（Neurology Teaching NeuroImages）

来源：《Neurology》杂志 Teaching NeuroImages 精选，175案例中筛选出**28个高特异性"影像模式→诊断"对子**，分6组：

| 组别 | 涵盖病种 |
|------|---------|
| I1. 遗传/代谢性脑白质病 | ALSP、GFAP星形胶质细胞病、NIID、成人型Alexander病、MLD、CTX、X-ALD额叶型、MLC、L2-HGA、DM1 |
| I2. 线粒体/代谢性 | MELAS（DWI-ADC不匹配）、LHON、GM1神经节苷脂贮积症 |
| I3. 炎症/自身免疫 | CAA-ri、神经结节病（髓静脉扩张）、MOGAD视神经炎、抗NMDAR脑炎 |
| I4. 感染性 | 神经莱姆病（脑桥猫头鹰眼）、日本脑炎、脊髓髓内囊虫病 |
| I5. 血管性/卒中 | 双侧旁正中延髓梗死（心形征）、椎动脉V3段夹层（管壁MRI）、ALS运动带征 |
| I6. 特殊情境 | 散发型CJD、PML vs PML-IRIS（FDG-PET鉴别）、Joubert综合征、Wilson病、危重症脑微出血、ATTRm神经病 |

每条均含：影像标志物（序列）→ 指向诊断 → 确诊手段 → 核心误诊陷阱。

## 使用示例

**输入**：25 岁男性，腹泻 3 天后四肢无力 5 天，腱反射消失，病理反射(-)，肌电图传导速度减慢，CSF 蛋白升高细胞正常。

**框架输出**（节选）：

> 定位：腱反射消失+肌张力下降+病理反射阴性 → 下运动神经元，周围神经。EMG 传导速度减慢 → 脱髓鞘型。CSF 蛋白-细胞分离 → GBS 近乎病理诊断级别的特征。前驱腹泻 → Campylobacter 触发自身免疫反应。
>
> 诊断：GBS（AIDP 亚型），确诊级别。
>
> 紧急问题：已有呼吸力弱，监测 FVC，遵循 20-30-40 规则判断插管时机。
>
> 治疗：IVIG 或血浆置换；激素无效且有害，禁用。
